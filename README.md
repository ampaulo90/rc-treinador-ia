# RC Treinador Digital

Aplicativo para gerar treinos de corrida (exportar YAML para Garmin) e analisar CSVs do Garmin.

## Como usar

- **Gerar treino**: preencha os campos e baixe o arquivo .yaml
- **Analisar CSV**: envie um CSV exportado do Garmin e veja métricas e gráficos

## Deploy

Hospedado no Streamlit Cloud.
